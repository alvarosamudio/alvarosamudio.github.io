Visual Studio Code es el editor más popular entre desarrolladores. Con estos atajos y extensiones puedes duplicar tu productividad.

## Atajos de teclado imprescindibles

### Generales

| Atajo | Acción |
|-------|--------|
| `Cmd+P` | Abrir archivo por nombre |
| `Cmd+Shift+P` | Paleta de comandos |
| `Cmd+B` | Mostrar/ocultar barra lateral |
| `Cmd+J` | Mostrar/ocultar terminal |

### Edición

| Atajo | Acción |
|-------|--------|
| `Cmd+D` | Seleccionar siguiente ocurrencia |
| `Option+Up/Down` | Mover línea arriba/abajo |
| `Shift+Option+F` | Formatear documento |
| `Cmd+/` | Comentar/descomentar línea |

## Extensiones esenciales

- **Prettier**: Formateo automático de código
- **ESLint**: Detección de errores en JavaScript/TypeScript
- **GitLens**: Información contextual de Git
- **Material Icon Theme**: Iconos descriptivos para archivos
- **Live Share**: Colaboración en tiempo real

## Configuración recomendada

```json
{
  "editor.fontSize": 14,
  "editor.tabSize": 2,
  "editor.wordWrap": "on",
  "editor.formatOnSave": true,
  "editor.minimap.enabled": false,
  "workbench.colorTheme": "One Dark Pro"
}
```

Un editor bien configurado marca la diferencia en tu día a día como desarrollador.
